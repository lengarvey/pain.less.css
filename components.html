Components

Buttons

To make a button, just use the BUTTON tag. Add one of the standard classes (primary, success, warning, danger, info) to alter the button styles. 

<button>Default</button>
<button class="primary">Primary</button>
<button class="success">Success</button>
<button class="warning">Warning</button>
<button class="danger">Danger</button>
<button class="info">Info</button>

You also can use the .button class where you want to make a non-button element to look like a button. You also can combine any of the standard style classes in conjunction with the button class in order to modify the button styles.

Default Default Success Warning Danger Info 
<a href="#" class="button">Default</a>
<a href="#" class="button primary">Default</a>
<a href="#" class="button success">Success</a>
<a href="#" class="button wraning">Warning</a>
<a href="#" class="button danger">Danger</a>
<a href="#" class="button ingo">Info</a>

Input Buttons

<input type="button"> and <input type="submit"> elements also should work, but generally should be avoided as they're not really semantic, don't support icons and hard to style.

Element Groups

Whenever you need to create a group of elements, such as pills, and such, please use the .group class on the grouping element. 

<div class="group">
  <button>One</button>
  <button class="primary">Two</button>
  <button>Three</button>
  <button>Four</button>
</div>

You also can group input fields and select boxes with buttons. 

<div class="group">
  <input type="search" placeholder="Search">
  <button class="primary">Go</button>
</div>

Navigation

Navigation in the case of pain.less.css should be a combination of the NAV and the A elements. Like so. 

<nav>
  <a href="one">One</a>
  <a href="two">Two</a>
  <a href="three">Three</a>
</nav>

The actual look of the nav element is contextual. For example if you put the NAV element within the BODY > HEADER element (or add the .bar class to it), it will look like the navbar in the header.

Home Cheez Blog 
<body>
  <header>
    <a href="./" class="logo">Cheezburgerz!</a>

    <nav>
      <a href="./">Home</a>
      <a href="./cheez">Cheez</a>
      <a href="./blog">Blog</a>
    </nav>
  </header>

  .... the rest of the page markup
</body>

If you put the NAV component structure in the ASIDE element in on of the sidebars, or add the .list class to it, you'll end up with a vertical navigation list.

Home Cheez Blog 
<nav class="list">
  <a href="./" class="active">Home</a>
  <a href="./cheez">Cheez</a>
  <a href="./blog">Blog</a>
</nav>
