Code blocks

All the code blocks should be done with the <code> elements. You can either inline them in the text like so.

CSS is awesome! body { display: none; }

<p>CSS is awesome! <code class="css">body { display: none; }</code></p>

You also can embed the CODE tags into PRE to get blocks of code 

@media screen {
  .yourmama { font-weight: bold; }
  .yourmama { max-width: none;   }
}
<pre><code lang="css">
  @media screen {
    .yourmama { font-weight: bold; }
    .yourmama { max-width: none;   }
  }
</code></pre>

Lists

Lists (UL, OL, DL) are simply lists, just like our lord and savior Tim Berners-Lee envisioned them.

Some text before the list, a genuine bla bla bla-ish stuff. I feel like I'm totally making sense here.

  • Take a shower
  • Brush teeth
  • Shave the yack
  • Put the goof on
<ul>
  <li>Take a shower</li>
  <li>Brush teeth</li>
  <li>Shave the yack</li>
  <li>Put the goof on</li>
</ul>

You can make your lists horizontal, by adding the .horizontal css class on them

Some text before the list, a genuine bla bla bla-ish stuff. I feel like I'm totally making sense here.

  1. Take a shower,
  2. Brush teeth,
  3. Shave the yack,
  4. Put the goof on
<ol class="horizontal">
  <li>Take a shower</li>
  <li>Brush teeth</li>
  <li>Shave the yack</li>
  <li>Put the goof on</li>
</ol>
