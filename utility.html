Utility Items

As we're using the LESS preprocessor to drive the implementation, there is a number of utility classes/mixins that are available and can be reused in your application code.

NOTE: all the classes in this list can be used as stand alone css class on any element or as a mixin, for example. 

<div class="my-thing uppercase">Text</div>
.my-thing {
  .uppercase;
  // rest of the styles
}

Style Modifieres

By default pain.less.css provides several standard style modifiers that can be assigned to any element and used in conjunction with various components.

  • primary - the primary call for action
  • success - paints it green
  • warning - warning element (normally yellow)
  • danger - danger element (normally red)
  • info - information elements

Text Case Switch

You can use the uppercase, lowercase and capitalize class to change the elements text casing.

upperCASE

LOWERcase

capITalize

<p class="uppercase">upperCASE</p>
<p class="lowercase">LOWERcase</p>
<p class="capitalize">capITalize</p>
